API RESTful for study Python + Flask

Modelo relacional do BD para um sistema que tem como objetivo ajudar pessoas que sentem a necessidade de conversar e expor seus sentimentos com uma outra pessoa, de maneira que se sintam numa zona segura para compartilhar tudo que achar necessário. Dentro do sistema, vai existir os acolhedores que serão pessoas cadastradas e selecionadas por uma série de critérios pré-estabelecidos devido a possível fragilidade das pessoas que estão procurando ajuda. Para facilitar o encaminhamento das pessoas que precisam de especialistas, terá também uma lista com contatos de instituições que possam ajudar estas pessoas em determinadas situações que se encontram. Com o intuito de uma melhor organização, tera uma lista de acolhedores disponíveis no momento para entrar um chat, onde o mesmo terá um histórico registrando o que foi conversado entre o acolhedor e o usuário anônimo que está procurando ajuda.
